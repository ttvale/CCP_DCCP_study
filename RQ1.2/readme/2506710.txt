Description
-----------
overleap.js is a small JavaScript library to add blank space at
the bottom of a webpage. With overleap.js, your visitors can scroll/page
your website in a more native way.

Usage
-----
To use overleap.js, simply include the overleap.js into your HTML document's header:
```<script type="text/javascript" src="overleap.js"></script>```
and add a emtpy `div` with ID `overleapSpacer` at the bottom of your document:
```<div id="overleapSpacer"></div>```

References
