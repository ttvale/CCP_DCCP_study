This repository contains the gettext PO files for all Hackasaurus translations contributed to localize.mozilla.org that are at least 20% complete.

To update this repository, simply run:

    python gettranslations.py

To use this repository in an internationalized Hackasaurus project like the [goggles][] or the [website][], just point the project's `locale` directory to the root of this repository.

  [goggles]: https://github.com/hackasaurus/webxray
  [website]: https://github.com/hackasaurus/hackasaurus.org
