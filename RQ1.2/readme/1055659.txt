This repository contains various Nagios plugins and scripts, and some stuff for
Centreon (<http://www.centreon.com>):

 * auditcent

 * check_by_ssh_master - Execute commands with SSH master connections

 * check_linux_memory - Check free memory on Linux servers

 * check_snmp_cisco - Various checks for Cisco devices

 * check_snmp_hr_lm - Various Host Resources and LAN Manager SNMP checks

 * check_snmp_if - Check status of interfaces using SNMP (IF-MIB)

 * check_snmp_ups - Check status of UPSes via SNMP (UPS-MIB)

 * check_tsm - Perform Tivoli Storage Manager server checks

 * nagammu

 * nagios-schedule-downtime
