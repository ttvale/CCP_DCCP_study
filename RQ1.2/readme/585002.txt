(un)ZIP Module for Appcelerator Titanium
======================

DESCRIPTION
=======================
This module allows you to unzip files within titanium.
Zip support will be added soon.

Based upon the objective c class from http://code.google.com/p/ziparchive/ (licensed under MIT)
