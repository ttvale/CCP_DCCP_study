Plugin Name: Add Post TrackEvent
Plugin URI: http://xfeng.me/add-post-trackevent-wordpress-plugin/
Author: JoysBoy 
Author URI: http://xfeng.me
Description: This plugin automatically add the onclick="_gaq.push(['_trackEvent', 'category', 'action', 'URL']);" to link in a post. 
You can modify category and action that you like. You can choose opt_label as domain or url.
为你的博客文章内链接加上Google Analytics事件跟踪，你可以自己修改category和action名字，方便你统计文章内非本站链接点击。
本插件提供了两种模式，事件跟踪标签为完整的外链地址或仅为外链域名，你可以选择注释插件中的正则。
