Munin plugins to monitor Kannel. 

Supported stable Kannel versions: 1.4.0 and higher.

Plugins included:

- SMS throughput (sent/received message rate)
- SMS store size
- SMS queue size
- DLR throughput (sent/received DLR rate)
- DLR queue size

Configuration:

[kannel_*]
user nobody
env.STATUS_URL http://localhost:13000/status.xml
