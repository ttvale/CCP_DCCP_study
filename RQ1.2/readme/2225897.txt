This template is based on KDE Air by Weng Xuetian http://csslayer.tk
This use some css3 tag like border-image or text-shadow and css-transition.
The jquery plugin used are Backstrech for have a fullscreen background and cssAnimate for the animation of the banner.
Actually not support IE at 100%.

Module supported:
Download_Gallery
News


On setting of news Module edit:

Post Header with:
<table cellpadding="0" cellspacing="0" class="post_header">
<tr>
<td><h1>[TITLE]</h1></td>
<td rowspan="3" style="display: [DISPLAY_IMAGE]">[GROUP_IMAGE]</td>
</tr>
<tr>
<td class="public_info"><b>[TEXT_POSTED_BY] [DISPLAY_NAME] ([USERNAME]) [TEXT_ON] [PUBLISHED_DATE]</b></td>
</tr>
<tr style="display: [DISPLAY_GROUP]">
<td class="group_page"><a href="[BACK]">[PAGE_TITLE]</a> >> <a href="[BACK]?g=[GROUP_ID]">[GROUP_TITLE]</a></td>
</tr>
</table>
<div class="post_image">[PICTURE]</div>
Post footer with:
<p class="post_changed">[TEXT_LAST_CHANGED]: [MODI_DATE] [TEXT_AT] [MODI_TIME]</p>
<a href="[BACK]" class="post_back">[TEXT_BACK]</a>
