
Data export/import capabilities are very important in most of utility apps. In one of my tutorials I demonstrate how to implement a data import from a CSV (Comma Separated Values) file into an iOS app via email attachment.

This tutorial can be found here: 

1. http://notatkiprogramisty.blox.pl/2011/12/Importing-CSV-data-file-into-an-iOS-app-via-email.html

2. http://notatkiprogramisty.blox.pl/2011/12/Importing-CSV-data-file-into-an-iOS-app-via-email-1.html

and

1. http://spritebandits.wordpress.com/2011/12/14/importing-csv-data-file-into-an-ios-app-via-email-attachment/

2. http://spritebandits.wordpress.com/2011/12/21/importing-csv-data-file-into-an-ios-app-via-email-attachment-part-2/

Hope you’ll find it useful. Let me know if you have any questions or comments.
