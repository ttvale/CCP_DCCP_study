Remote control your Philips TV
==============================

This application uses a the JSON Rest API build in new philips TV's.
See http://jointspace.sourceforge.net/

How to install
--------------

Just download and extract to your webserver

How does it work
----------------

The application uses jQuery for maximum support in all browsers.
Your webbrowser will connect to the IP address you configure on port 1925 and uses HTTP to communicate with the TV.
