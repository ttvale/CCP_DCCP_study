# RngTng Textmate bundle

This is a Textmate bundle for personal use.

## Features
It comes with some nice enhancements:

 * trim + save
   this allows to trim trailing spaces and saveing file with just on shortcut (shit+commdand+s)
 * Align assignments
 * Highlight trailing white spaces
   -> http://remysharp.com/2008/03/30/trailing-white-space-in-textmate/
 * Metal (Nabaztag Bootcode) syntax highlight


## INSTALL

    cd ~/Library/Application\ Support/TextMate/Bundles/
    git clone
