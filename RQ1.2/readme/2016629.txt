Conditionally rollout features with Javascript
=====

Description
--------

That is a wrapper for the project https://github.com/jamesgolick/rollout
that integrates into your Rails app and make it accessable in your
Javascript code.

Installation
-------

Add it to your Gemfile:

``` ruby
gem 'rollout'
gem 'rollout-js'
```
