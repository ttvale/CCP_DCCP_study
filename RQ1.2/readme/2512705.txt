WCA Competitions
================

Prerequisites
-------------

* Ruby >= 1.9.3
* Bundler

How to get it running
---------------------

    git clone https://github.com/timhabermaas/wca-competitions.git
    cd wca-competitions
    bundle
    bundle exec rails server
