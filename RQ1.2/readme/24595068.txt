Hash::Map version 0.016
=======================

Manipulate hashes map like.

INSTALLATION

To install this module, run the following commands:

    perl Makefile.PL
    make
    make test
    make install

DEPENDENCIES

This module requires these other modules and libraries:

    Carp
    Clone
    Exporter::Tiny
    Scalar::Util
    Try::Tiny
