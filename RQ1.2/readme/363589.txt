# mayancal

Clojure program to generate an illustrated Mayan calendar
(as a PDF file) for the specified year (default 2012).

## Usage

java -jar mayancal.jar -bn outfile-basename [--year 19xx | 20xx]

Example: java -jar mayancal.jar -bn /tmp/mcal84 --year 1984
         Generates a Mayan calendar for 1984 into /tmp/mcal84.pdf
