# Keyboard Scroll jQuery Plugin

A simple jQuery plugin for keyboard scrolling through different sections in a webpage.

*Note: still in alpha (first github repo, any help and suggestions are appreciated).*

##Example:

* included in the github repo
* [expressoviagens.com](http://expressoviagens.com/ "Expresso Viagens")
