USE THE UPSTREAM GIT REPO INSTEAD http://root.cern.ch/git/root.git or here https://github.com/root-mirror/root. Mirror of CERN's ROOT svn repository, updates are run hourly. master branch is svn trunk. Branch branches/dev is suppossed to be a set of branches instead of folders, but you can still find the branches in branches/dev and patch branches under branches/. Tags are under tags/.
